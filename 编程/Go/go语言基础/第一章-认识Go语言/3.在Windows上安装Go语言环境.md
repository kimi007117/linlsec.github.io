# 在Windows上安装Go语言环境

### 安装包下载：

**https://golang.google.cn/dl/**

![Windows-install-go](/Users/linlsec/Desktop/linlsec.github.io/images/编程/Windows-install-go.png)

### 查看是否安装成功

```shell
go version
```

### 设置环境变量

```shell
GOROOT C:\Go  # go安装路径
GOPATH C:\Go\bin
```

### 查看Go环境变量

```shell
go env
```
